About the Player

The app has been initialized as soon as the document was ready! Read the following instructions on how to create a new song and "play"/"pause"/"stop" it within the console.

Sorry to get your hopes up, but no music actually plays here. This "app" is simply a javascript exercise exhibiting my competence within the language.

INSTRUCTIONS: (withinConsole)

Create A Song & addToLibrary:

musicPlayerApp.addToLibrary('title', 'artist', 'album', coverartURL);

removeFromLibrary:

musicPlayerApp.removeFromLibrary('title');

addToPlaylist:

musicPlayerApp.addToPlaylist('title');

removeFromPlaylist:

musicPlayerApp.removeToPlaylist('title');

shufflePlaylist

musicPlayerApp.shufflePlaylist();

playSong

musicPlayerApp.playSong('title');

pauseSong

musicPlayerApp.pauseSong();

playnextSong

musicPlayerApp.playnextSong();
